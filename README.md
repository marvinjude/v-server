## Setup

```
yarn
```

## Populate Database

```
yarn populate-db
```

## Development

Run dev server without delay

```
yarn dev
```

To start the dev server with some response delay which defaults to `2000ms` run:

```
yarn dev:delay
```

You can change the default delay in `package.json`
